# Deployment

Required environment variables:
- `GEMINI_API_KEY` or `OPENAI_API_KEY`
- `GEMINI_MODEL` or `CHAT_MODEL`
- `NEXT_PUBLIC_APP_URL`

Future production variables:
- `DATABASE_URL`
- Supabase URL/key variables
- Storage bucket
- Payment provider keys such as Razorpay

Run locally:
```bash
npm install
npm run dev
```
