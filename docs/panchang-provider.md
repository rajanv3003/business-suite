# Panchang Provider

The app must not scrape a single Panchang website directly.

Production provider interface should support:
- Demo/mock data
- Configurable production API
- Admin CSV/JSON upload
- Manually reviewed editorial events
- Cached city/timezone records

If verified data is missing, UI and generated outputs must say: "इस दिन का verified Panchang data उपलब्ध नहीं है।"
